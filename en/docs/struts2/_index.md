While the framework is simple to use, creating non-trivial applications assumes a working knowledge of many JEE technologies, including: 

 - Java
 - Filters
 - JSP
 - Tag Libraries
 - JavaBeans
 - HTML and HTTP
 - Web Containers (such as Tomcat)
 - XML
