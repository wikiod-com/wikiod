EPPlus is a .NET library that reads and writes Excel 2007/2010/2013 files using the Open Office Xml format (xlsx). 

EPPlus supports:

 - Cell Ranges
 - Cell styling (Border, Color, Fill, Font, Number, Alignments)
 - Charts
 - Pictures
 - Shapes
 - Comments
 - Tables
 - Protection
 - Encryption
 - Pivot tables
 - Data validation
 - Conditional formatting
 - VBA
 - Formula calculation
