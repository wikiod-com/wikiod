## Acrônimos relacionados à rede
Observe que alguns termos como JIT e GC são genéricos o suficiente para se aplicarem a muitos ambientes de linguagem de programação e tempos de execução.

CLR: Common Language Runtime
IL: Idioma Intermediário
EE: mecanismo de execução
JIT: Compilador just-in-time
GC: Coletor de Lixo
OOM: Sem memória
STA: apartamento single-thread
MTA: apartamento multi-thread
