JIT derlemesi veya tam zamanında derleme, kodun veya önceden derlemenin yorumlanmasına alternatif bir yaklaşımdır. .NET çerçevesinde JIT derlemesi kullanılır. CLR kodu (C#, F#, Visual Basic, vb.) önce Yorumlanan Dil veya IL adı verilen bir şeye derlenir. Bu, makine koduna daha yakın olan ancak platforma özel olmayan daha düşük seviyeli bir koddur. Bunun yerine, çalışma zamanında bu kod, ilgili sistem için makine koduna derlenir.

JIT derlemesini neden kullanmalısınız?
- Daha iyi uyumluluk: Her CLR dili, IL için yalnızca bir derleyiciye ihtiyaç duyar ve bu IL, makine koduna dönüştürülebileceği herhangi bir platformda çalışabilir.
- Hız: JIT derlemesi, önceden derlenmiş kodu çalıştırma hızı ile yorumlama esnekliğini birleştirmeye çalışır (derlemeden önce potansiyel optimizasyonlar için yürütülecek kodu analiz edebilir)

Genel olarak JIT derlemesi hakkında daha fazla bilgi için Wikipedia Sayfası: https://en.wikipedia.org/wiki/Just-in-time_compilation

## IL derleme örneği
Basit Merhaba Dünya Uygulaması:

    using System;
    
    namespace HelloWorld
    {
        class Program
        {
            static void Main(string[] args)
            {
                Console.WriteLine("Hello World");
            }
        }
    }
